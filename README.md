# React Quiz App 

A fast, interactive web programming quiz built from scratch using React and Vite. This application tests knowledge across 100 questions covering fundamental JavaScript, Object-Oriented Programming (OOP), the DOM, NPM, and React.

##  Features
* **Topic Filtering:** Choose to test yourself on a specific category or tackle all 100 questions at once.
* **Dark Mode  / Light Mode :** Smooth, system-aware theme toggling for comfortable late-night coding sessions.
* **Countdown Timer:** Adds a bit of pressure! The timer adjusts automatically based on the number of questions selected.
* **Post-Quiz Review:** At the end of the quiz, review the questions you got wrong along with detailed explanations for the correct answers.
* **Responsive Design:** Built with modern CSS variables, ensuring a clean and accessible UI on any device.

## Tech Stack
* **Framework:** React 18
* **Build Tool:** Vite
* **Styling:** Custom CSS (No external libraries like Bootstrap or Tailwind)
